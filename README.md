# web-flower
author: Xuan Bui

Cài đặt phụ thuộc: xampp, database

Công nghệ sử dụng:
  - Front-end: Bootstrap, CSS
  - Back-end: PHP, JQuery
  
 
 Mô tả chức năng:

    - Giúp nhà bán hàng quảng bá tiếp thị sản phẩm đến tay người tiêu dùng, CRUD sản phẩm, users, đơn hàng
